Cloud Computing: Dining Concierge Bot

Created a ChatBot using the Amazon Lex Service. Based on the parameters collected from the user, pushed the information collected (location, cuisine, etc.) to an SQS queue. Collected Restaurant data from the Yelp API which is later on used for Recommendation and stored it in a DynamoDB Table. Based on a conversation with the customer, the LEX chatbot will identify the customer’s preferred ‘cuisine’. We then search through ElasticSearch to get random suggestions of restaurant IDs with this cuisine. At this point, we also query the DynamoDB table with these restaurant IDs to find more information about the restaurants we want to suggest to the customers like name and address of the restaurant.
